create demo (

#### 索引语法

- 创建索引
```sql
create [unique | fulltext] index index_name on table_name (column_name[, ...]);
```

- 查询表是否有索引
```sql
show index from goods\G
show indexes from goods\G   -- \G 做结束符: 以垂直输出的形式输出结果
```

- 删除索引
```sql
drop index index_name on table_name;
```
